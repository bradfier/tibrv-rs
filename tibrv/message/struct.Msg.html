pub struct Msg { /* fields omitted */ }

A struct representing an owned Rendezvous Message.

The memory allocated to this type of Message is the responsibility of the Rust application. When this type is dropped, tibrvMsg_Destroy will be run to free any memory allocated to store the message.

Add a MsgField to this message.

The referenced field must be marked mutable, as although the process should not mutate the field, the C library makes no guarantee.

The contents of message fields are always copied, therefore slice types must be Copy. A borrowed MsgField does not need to live beyond the point where it is added to the Msg.

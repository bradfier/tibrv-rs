Encode a u16 as an IP Port message field.

Rendezvous has special provisions for network data types, IPv4 Addresses and IP Ports are encoded as special types using Network Byte Ordering.

Since we already provide an Impl for 'normal' u16 this function will encode using the special byte ordering.
